# Initial Setup
1. From parent directory, Install Vue, might take a few seconds to trigger the prompt
   - Command: 
    ```npm create vite@latest```
   - Might get prompted to install create vite with latest version
   - `npm -v` was at `9.6.7`
2. Prompted for project name: vue3-travel-app
3. Prompted for framework: Vue
4. Prompted for flavor: Javascript
5. Opened newly created file on vscode
6. Install dependencies: `npm install`
7. Run app: `npm run dev`

# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.
